# landsoul

Drop in css library landsoul  
Support dark mode, use `prefers-color-scheme: dark` or `body[data-theme="dark"]`  
IE11 is not supported  
normalize.css is optional  
Preview: <https://yieldray.github.io/landsoul/>

## dev

```bash
$ npm install
$ npm run dev
# xdg-open index.html
```

## build

```bash
$ npm run build
```
